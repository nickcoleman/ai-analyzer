## WRITING GUIDELINES FOR EXECUTIVE SUMMARY

### Recommendation Language

**DO:**
- "BUY" (not "consider buying", not "might be attractive")
- "HOLD" (not "neutral", not "wait and see")
- "SELL" (not "reduce position", not "might be at risk")
- Pair with confidence: "BUY (82% confidence)"

**DON'T:**
- Use hedging language ("could be", "may be", "might be")
- Use qualitative terms ("undervalued", "attractive") without context
- Leave ambiguity about whether to act

### Confidence Score Interpretation

- 95-100%: Maximum conviction (all thesis drivers confirmed, backtests passed, catalysts imminent)
- 80-94%: High conviction (most drivers confirmed, backtest passed)
- 65-79%: Moderate-to-high conviction (thesis sound but some uncertainty remains)
- 50-64%: Moderate conviction (balanced risk/reward but not clear winner)
- Below 50%: Don't recommend (wait for more clarity)

### Price Target Format

**REQUIRED:**
- Three scenarios: Bull (upside), Base (most likely), Bear (downside)
- Each with specific price and percentage return
- Each with explicit probability weight
- Not ranges like "$145-150" but specific scenarios with different probabilities

**Example (CORRECT):**
```
BASE CASE: $142 (+35%)  [55% probability]
BULL CASE: $165 (+65%)  [25% probability]  
BEAR CASE: $115 (-15%)  [20% probability]
```

**Example (INCORRECT):**
```
Price Target: $145-150  [not specific enough, no probabilities]
Upside Target: $165    [what's the probability? base case or bull?]
```

### Action Verb Precision

**Your Action Field Must Start With:**
- "BUY X% at $[price] immediately" (not "consider buying")
- "ADD X% on pullback to $[price]" (not "might add on weakness")
- "SELL/EXIT position" (not "consider reducing")
- "HOLD X%, wait for $[trigger]" (not "hold and see")

### Current Position Status Icons

```
‚Üë = Position up (green, doing well)
‚Üí = Position flat (neutral)
‚Üì = Position down (red, concerning)
- = No position yet
```

## QUALITY CHECKLIST FOR EXECUTIVE SUMMARY

### Must Pass (Hard Requirements)

- ‚úÖ Recommendation is **first** (not buried)
- ‚úÖ Recommendation is **one word**: BUY, HOLD, or SELL
- ‚úÖ Confidence % is **explicit** (not "moderate confidence")
- ‚úÖ Price targets are **specific** (e.g., "$21.67" not "$20-25")
- ‚úÖ Expected return is **calculated** (e.g., "+35.8%" not "upside potential")
- ‚úÖ Current price is **shown** with date (e.g., "$15.96 as of Nov 24")
- ‚úÖ Three scenarios with **explicit probabilities** (sum to 100%)
- ‚úÖ Action verb is **clear and immediate** (BUY/ADD/HOLD/SELL)
- ‚úÖ Entry price is **specific** (e.g., "$15.96-16.20" not "near support")
- ‚úÖ Position size is **stated** as % of portfolio (not "moderate")
- ‚úÖ Time horizon is **explicit** (not "long-term")
- ‚úÖ Three thesis drivers are **specific** (with numbers if available)
- ‚úÖ Three decision triggers are **actionable** (user would recognize immediately)

### Tone & Language

- ‚úÖ Professional but direct (not academic, not casual)
- ‚úÖ Confident but not overconfident (reflect actual conviction)
- ‚úÖ Specific but not over-detailed (save details for sections 2-4)
- ‚úÖ Action-oriented (what user should **do**, not what **might happen**)
